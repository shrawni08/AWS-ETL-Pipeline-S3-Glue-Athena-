🚀 AWS ETL Pipeline using S3, Glue & Athena
📌 Overview

This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline using AWS services.

The pipeline:

Stores raw data in S3
Uses Glue Crawler to infer schema
Stores metadata in Glue Data Catalog
Performs transformation using Glue Visual ETL
Writes processed data in Parquet format to S3
Uses Athena for querying
